# 大一下学期课程资料

## 课程简介

- 开课学院：电信学院
- 学期：大一下学期  
- 主要内容：高等数学进阶、物理基础、数据结构、概率统计、电路基础

**资料来源: [谢悦晋的GitHub仓库](https://github.com/1363033313/First-year-down.git)**
