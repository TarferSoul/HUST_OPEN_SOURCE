# 大二下学期课程资料

## 课程简介

- 开课学院：电信学院
- 学期：大二下学期
- 主要内容：信号系统、信息论、数理方程、计算机组成、随机过程

## GitHub资源库

**资料来源: [TarferSoul的GitHub仓库](https://github.com/TarferSoul/Second-semester-of-sophomore-year.git)**

- **备注**: 电信专业核心课程，为通信、控制等后续课程打基础
